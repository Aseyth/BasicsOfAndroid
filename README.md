## Retrofit 2.1

Type-safe HTTP client for Android and Java by Square, Inc.

### Prerequisites

Add thoses line to your build.gradle (Project)

```
compile 'com.squareup.retrofit2:retrofit:2.1.0'
compile 'com.squareup.retrofit2:converter-gson:2.1.0'
```
